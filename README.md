<h1>AI Enabled Incident Response Automation in Microsoft Azure</h1>


<h2>Overview</h2>
The project involves development of a ChatGPT solution on the Azure Cloud platform to enhance Cybersecurity incident management.
<br />

<h2>Environments Used </h2>

- <b>Microsoft Azure</b>

<h2>To-Do</h2>

To successfully complete the project, we need to perform following tasks:

1.	Create Free Azure Subscription
2.	Create an Automation Rule in Microsoft Sentinel
3.	Generate a New Secret Key from OpenAI
4.	Create Playbook for ChatGPT from Scratch 
5.	Assign Appropriate Privileges to Microsoft Sentinel
6.	Run ChatGPT on Cybersecurity Incidents
7.	Create a New Automation Rule to Fully Automate Cybersecurity Incidents as soon as It’s Triggered
